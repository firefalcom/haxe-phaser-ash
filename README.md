# haxe-phaser-ash

haxe-phaser-ash (hpa) is a light framework combining the [Ash framework](https://github.com/nadako/Ash-Haxe) and the externs for [Phaser](https://github.com/photonstorm/phaser).

## Usage

### Dependencies

```shell
haxelib install ash
haxelib install phaser
```
